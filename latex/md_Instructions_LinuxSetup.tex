Follow \href{https://docs.github.com/en/authentication/connecting-to-github-with-ssh/generating-a-new-ssh-key-and-adding-it-to-the-ssh-agent}{\texttt{ these instructions}} to create an SSH key. When asked what to name the file, press enter. This will give the file the default name. When asked to create a password, press enter. You may stop when you reach the section titled \char`\"{}\+Generating a new SSH key for a hardware security key\char`\"{}. To add the key to your account, follow \href{https://docs.github.com/en/authentication/connecting-to-github-with-ssh/adding-a-new-ssh-key-to-your-github-account}{\texttt{ these directions}}, starting in the \char`\"{}\+Adding a new SSH key to your account\char`\"{} section. To validate this was successful, run the following command\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{ssh git@github.com}

\end{DoxyCode}
